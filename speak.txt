use the openssl library
在電腦網路上，OpenSSL是一個開放原始碼的軟體函式庫套件，應用程式可以使用這個套件來進行安全通訊
，避免竊聽，同時確認另一端連線者的身分。這個套件廣泛被應用在網際網路的網頁伺服器上。

use the native windows secure channel library  意思

安全通道，也稱為Schannel，是一種安全支持提供程序（SSP），它包含一組通過加密提供身份認證和安全的私人通信的安全協議。
Schannel主要用於需要安全超文本傳輸協議（HTTP）通信的Internet應用程序。

Checkout Windows-style, commit Unix-style line endings

讓 Git 在 Checkout 時自動將 LF（Unix）轉換為 CRLF（Windows）；在 Commit 時自動將 CRLF（Windows）轉換為 LF（Unix）。若用於跨平台專案，建議選擇此項。

? Checkout as it is, commit Unix-style line endings

設定 Git 在 Checkout 時不自動轉換；在 Commit 時自動將 CRLF（Windows）轉換為 LF（Unix）。若用於跨平台專案，建議於 Unix 主機上選擇此項。


? Checkout as it is, commit as it is

設定 Git 在 Checkout 和Commit 時皆不自動轉換。不建議用於跨平台專案。
